Is between

Write a program that reads three integer numbers and prints true if the first number is between the second and the third one (inclusive). Otherwise, it is to print false.

The sorting order of the two last arguments can be any.


Sample Input:
3 3 3

Sample Output:
true


Sample Input:
40 30 50

Sample Output:
true


Sample Input:
40 100 20

Sample Output:
true



Memory limit: 256 MB
Time limit: 8 seconds
